# Update[](#actualizacion "Link to this heading")

1.  In the Windows Control Panel, open the Add or remove programs window. Uninstall all Qflow components. These are:
    
    > 1.  Qflow Backend services
    >     
    > 2.  Qflow BPA
    >     
    > 3.  Qflow BPM
    >     
    > 4.  Qflow Design
    >     
    > 5.  Qflow Team
    >     
    > 6.  Qflow Web Services API
    >     
    > 7.  Qflow Task
    >     
    > 8.  Qflow Admin
    >     
    > 9.  Qflow Access
    >     
    
2.  Access the new version’s installer and run Setup.exe with administrator privileges.
    
    1.  Run the database installer
        
        1.  Select Update Qflow database
            
        2.  Make sure to correctly indicate the requested parameters
            
    2.  Install the backend’s WebAPI
        
    3.  Install Qflow Task (end users site)
        
    4.  Install REST web services (optional)
        
    5.  Install Qflow Design (Business Process Modeler)
        
    6.  Install Qflow Team (Organizational Model Manager)
        
    7.  Install Qflow Admin (System Administrator and Monitor)
        
    8.  Install Qflow Access (system access portal)
        
    9.  Finally, you can optionally install the business process modeler (BPM) and the business process administrator (BPA). For this, you will need to open a console in administrator mode and run the corresponding .msi.
