# Previous tasks[](#tareas-previas "Link to this heading")

1.  Carefully read the Qflow update section of the [Installation and Configuration of Qflow](02-InstallationAndConfiguration.html) manual. There you will find detailed instructions on how to update Qflow. This is a complementary document.
    
2.  Stop all Qflow services.
    
3.  Perform a database backup.
    
4.  Back up the Qflow web sites and services folders:
    
    > 1.  <%ProgramFiles%>\\Urudata, in which <%ProgramFiles%> must be replaced by the folder where the programs are installed on the computer with Qflow installed.
    >     
    > 2.  <%wwwroot%>\\QflowTask, <%wwwroot%>\\QflowDesign, <%wwwroot%>\\QflowTeam, <%wwwroot%>\\QflowAdmin, where <%wwwroot%> is the folder where web applications are installed by default. This folder is normally located at: “c:\\inetpub\\wwwroot”. If the WebForms site was installed, you will also need to back up the <%wwwroot%>\\QflowWebSite folder. It should be noted that the names of the folders may vary if this was changed during Qflow’s installation.
    >
