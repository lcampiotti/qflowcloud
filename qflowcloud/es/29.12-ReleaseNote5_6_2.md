# v5.6.2[](#v5-6-2 "Link to this heading")

## Resumen de características[](#resumen-de-caracteristicas "Link to this heading")

Seguimos mejorando **Qflow** para que tus procesos sean más sencillos de gestionar, potenciando el alcance de tu negocio. En esta versión, rediseñamos componentes clave, mejoramos conectores y sumamos funcionalidades que te van a ahorrar tiempo y clics.

¡Explora las novedades!

## Nuevo conector con ILovePDF[](#nuevo-conector-con-ilovepdf "Link to this heading")

Se incorporaron tres nuevas funcionalidades mediante conectores con la herramienta ILovePDF:

*   Convertir archivo Office: selecciona un archivo de Microsoft Office (Word, Excel, PowerPoint, etc.) que forme parte del proceso y conviértelo automáticamente a PDF.
    
*   Convertir imagen a PDF: permite seleccionar una imagen que sea parte del proceso y convertirla a PDF.
    
*   Unir archivos PDF: selecciona múltiples archivos PDF dentro del proceso para unirlos en un único documento.
    

Estas funciones amplían las posibilidades de automatización de documentos en los procesos sin necesidad de herramientas externas adicionales.

## Mejoras en el conector con Microsoft Word[](#mejoras-en-el-conector-con-microsoft-word "Link to this heading")

El conector con Microsoft Word incorpora un nuevo formato de etiquetas que Qflow reconoce y reemplaza automáticamente con datos o roles del proceso. Gracias a esta mejora:

*   Ya no es necesario completar manualmente la sección “Valores para reemplazar las etiquetas” en el panel de configuración del conector.
    
*   Se puede controlar cómo se muestran los datos con estructura compleja, como fechas y fechas con hora. Así, puedes incluir fechas formateadas directamente en la plantilla.
    

Más información en la [documentación del conector con Microsoft Word](36.14-MicrosoftWordConnector.html).

## Rediseño del componente de datos tipo documento[](#rediseno-del-componente-de-datos-tipo-documento "Link to this heading")

Renovamos el componente de datos tipo documento en los formularios de **Qflow Task** para que sea más práctico y fácil de usar.

*   Ahora, desde el mismo componente, es posible ver el valor actual, subir un nuevo archivo o abrir el documento seleccionado en una nueva pestaña (siempre que el navegador lo permita, como PDF e imágenes).
    
*   Se simplificó el uso de los datos de tipo documento. Ya no es necesario configurar la visibilidad del dato y ni de los adjuntos por separado: si el dato está visible en el formulario, puede abrirse, y si su valor es editable, también puede subirse un nuevo documento.
    
*   Esto también permite simplificar formularios, prescindiendo de la tabla de adjuntos en formularios que solo utilizan documentos como datos.

## Nuevo tipo de conexión para conectores de Outlook[](#nuevo-tipo-de-conexion-para-conectores-de-outlook "Link to this heading")

¡Ahora puedes usar tu cuenta de Microsoft para agendar tus citas en Outlook desde Qflow!

A partir de esta versión, puedes usar tu cuenta de Microsoft para conectar tus procesos de Qflow con Outlook, para una integración más simple y segura, sin necesidad de configuraciones adicionales complejas.

## Rediseño de mensajes de confirmación en Qflow Task[](#rediseno-de-mensajes-de-confirmacion-en-qflow-task "Link to this heading")

Se actualizaron los estilos de los mensajes de confirmación que se muestran al iniciar un proceso o responder una tarea en Qflow Task.

Además, es posible configurar el mensaje que se muestra, utilizando datos e información del proceso para brindar una mejor experiencia de usuario.
